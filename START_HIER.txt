🎮 TYPISCHER ABLAUF
═════════════════════════════════════════════════════════════════════════════════

  1. Öffne family-places.html
  2. Schau die Karte an
  3. Klick einen Marker (🌳 🍽️ 🎨 ⚽)
  4. Info-Panel öffnet sich
  5. Klick [Login] oder [Register]
  6. Gib Daten ein (Name, Email, Passwort)
  7. Klick [Bewerten]
  8. Bewerte für jede Altersgruppe (1-5 ⭐)
  9. Schreib optional einen Kommentar
  10. Klick [Bewertung absenden]
  11. ✅ Fertig! Deine Bewertung ist sichtbar
  12. Klick ❤️ um als Favorit zu speichern


⚙️ OPTIONAL: SERVER STARTEN
═════════════════════════════════════════════════════════════════════════════════

  Falls du den Backend-Server starten willst:

  1. Installiere Node.js (von nodejs.org)
  
  2. Öffne PowerShell im Projektordner
  
  3. Führe aus:
     npm install
  
  4. Starte den Server:
     npm start
  
  5. Öffne im Browser:
     http://localhost:3000/family-places.html

  Der Server speichert Daten dann persistent!

❓ HÄUFIGE FRAGEN
═════════════════════════════════════════════════════════════════════════════════

  F: Kann ich sofort starten?
  A: JA! Öffne einfach family-places.html im Browser!

  F: Brauche ich Node.js?
  A: Nein! Frontend funktioniert ohne Node.js
     Node.js nur optional für Backend/Server

  F: Wo werden meine Daten gespeichert?
  A: Im Browser (LocalStorage) - auf deinem Computer
     Mit Backend: Auf dem Server

  F: Kann ich neue Orte hinzufügen?
  A: Ja! Bearbeite family-places-app.js → samplePlaces Array

  F: Wie ändere ich Farben?
  A: Bearbeite family-places-styles.css → :root Variablen

  F: Kann ich die Karte auf eine andere Stadt ändern?
  A: Ja! Siehe PROJECT_OVERVIEW.md Sektion "Konfiguration"


🆘 PROBLEME?
═════════════════════════════════════════════════════════════════════════════════

  "Seite lädt nicht"
  → Prüf ob family-places.html existiert
  → Rechtsklick → Mit Browser öffnen

  "Karte zeigt nichts"
  → Prüf Internet-Verbindung
  → Lade die Seite neu (F5)

  "Marker nicht sichtbar"
  → Zoom rein/raus mit Mouse-Wheel
  → Verschiebe die Karte

  "Login funktioniert nicht"
  → Alle Felder müssen ausgefüllt sein
  → Probier mit anderen Daten
